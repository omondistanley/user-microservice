Sprint 1 (Backend Foundation, 1 week)

Ticket: P1-S1-T1
Title: Recurring Expenses Schema
Status: DONE
Type: DB
DB Tasks:
- Add table expenses_db.recurring_expense with:
  recurring_id (uuid pk), user_id (int), amount (numeric), currency (char(3)),
  category_code (smallint), category_name (varchar), description (varchar),
  recurrence_rule (weekly|monthly|yearly), next_due_date (date),
  last_run_at (timestamptz), last_created_expense_id (uuid),
  is_active (bool), created_at, updated_at.
- Add indexes:
  idx_recurring_user_active_due(user_id, is_active, next_due_date),
  idx_recurring_user(user_id).
Acceptance Criteria:
- Migration is idempotent.
- Fresh DB and existing DB apply migration successfully.

Title: Income Schema
Status: DONE
Type: DB
