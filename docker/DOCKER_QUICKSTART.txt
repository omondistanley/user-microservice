Docker Quick Start
==================

1) Copy env defaults (Compose reads project-root .env):

   cp .env.example .env

2) Build frontend static assets (user image copies frontend/):

   ./scripts/bootstrap-frontend.sh

3) Start the stack:

   docker compose up --build

4) Endpoints:

   - App (use this in browser):  http://localhost:8080   (API gateway + proxied UI)
   - User service (direct):      http://localhost:8000
   - Expense API:                 http://localhost:3001
   - Budget API:                  http://localhost:3002
   - Investments API:             http://localhost:3003
   - Postgres:                    localhost:5432

Migrations run automatically through one-off services (see docker-compose.yml).

Smoke checks:

   curl -s http://localhost:8080/health
   curl -s http://localhost:8000/health
   curl -s http://localhost:3001/health

Expo Go: in mobile/, cp .env.example .env and set EXPO_PUBLIC_GATEWAY_URL to http://<LAN-IP>:8080
