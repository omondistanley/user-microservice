Push this project to GitLab (dimcked94-group/expense-tracker)
============================================================

1. Create a Personal Access Token
   - Go to: https://gitlab.com/-/user_settings/personal_access_tokens
   - Create a token with scope: write_repository (or api)
   - Copy the token (you won't see it again)

2. Push using the script (no credentials stored in git config)
   From the project root (expense_tracker):

   GITLAB_USER=your_gitlab_username GITLAB_TOKEN=your_token ./push-gitlab.sh

   Replace your_gitlab_username with your GitLab login (must have push access to
   dimcked94-group/expense-tracker). Replace your_token with the token from step 1.

3. Or push manually (Git will prompt for username and password)
   Use your GitLab username and the token as the "password":

   git push -uf gitlab main

Remotes: origin = GitHub, gitlab = GitLab (HTTPS). Both point to main.
